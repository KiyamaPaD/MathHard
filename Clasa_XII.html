Clasa a XII‑a

Această pagină prezintă conceptele și formulele esențiale din materia de matematică pentru Clasa a XII‑a.

Capitolul Unu – Grupuri

  • 1. Legi de compoziție

    Concept: o lege de compoziție este o funcție binară care asociază fiecărei perechi de elemente un element din aceeași mulțime.

  • 2. Parte stabilă. Exemple

    Concept: identificarea componentei stabile (elementul invariant) în operațiile de compoziție.

  • 3. Comutativitatea legilor de compoziție

    Formulă: a • b = b • a pentru orice a, b din grup, dacă legea este comutativă.

  • 4. Asociativitatea legilor de compoziție

    Formulă: (a • b) • c = a • (b • c) pentru orice a, b, c.

  • 5. Elementul neutru al unei legi de compoziție

    Definiție: există un element e astfel încât e • a = a • e = a, ∀a.

  • 6. Elemente simetrizabile

    Definiție: pentru fiecare a există un element a⁻¹ astfel încât a • a⁻¹ = e.

  • 8. Noțiunea de grup

    Definiție: un grup este o mulțime G dotată cu o operație • care este asociativă, are element neutru și fiecare element are invers.

  • 10. Grupuri de matrice

    Concept: mulțimi de matrici care, cu operația de înmulțire sau adunare, formează grupuri.

  • 11. Grupul matricelor pătratice cu operația de înmulțire

    Condiție: matricea trebuie să fie inversabilă, iar operația de înmulțire este asociativă.

  • 12. Grupul matricelor pătratice cu operația de adunare

    Formulă: adunarea matricelor se efectuează element cu element; elementul neutru este matricea nulă.

  • 14. Clase de resturi modulo n

    Definiție: [a] = { x ∈ ℤ | x ≡ a (mod n) }.

  • 15. Grupuri de clase de resturi modulo n

    Concept: mulțimea claselor de resturi modulo n formează un grup cu operația de adunare modulo n.

  • 16. Reguli de calcul într-un grup. Ridicarea la putere

    Formulă: a^n = a • a • … • a (n factori); se aplică proprietățile grupurilor.

  • 17. Legi de simplificare într-un grup

    Concept: dacă a • b = a • c, atunci b = c, în grupurile în care există invers.

  • 18. Morfism și izomorfism de grupuri

    Definiții: un morfism de grupuri este o funcție f: G → H care respectă operația; izomorfismul este un morfism bijectiv.

  • 20. Noțiunea de inel

    Definiție: un inel este o mulțime cu două operații (adunare și înmulțire) care respectă proprietăți specifice (adunare este un grup abelian, înmulțirea este asociativă și se distribuie peste adunare).

  • 21. Inele numerice

    Exemple: ℤ, ℚ, ℝ.

  • 22. Inele de matrice

    Concept: mulțimea matricelor de un anumit ordin, cu adunare și înmulțire, formează un inel.

  • 23. Inelul claselor de resturi modulo n

    Definiție: ℤₙ, inelul format din clasele de resturi modulo n.

  • 24. Noțiunea de corp

    Definiție: un corp este un inel în care fiecare element nenul este inversabil la înmulțire.

  • 25. Forma algebrică a unui polinom

    Forma: P(x) = aₙxⁿ + ... + a₁x + a₀.

  • 26. Valoarea și rădăcina unui polinom

    Concept: P(x₀) = 0, atunci x₀ este rădăcina polinomului P(x).

  • 27. Gradul unui polinom și egalitatea polinoamelor

    Concept: gradul este cel mai mare exponent al lui x; două polinoame sunt egale dacă au aceleași coeficienți.

  • 28. Adunarea și înmulțirea polinoamelor

    Operații efectuate termen cu termen.

  • 29. Teorema împărțirii cu rest a polinoamelor

    Formulă: pentru polinoamele P(x) și divisorul D(x), există polinoame Q(x) și R(x) astfel încât P(x) = D(x)·Q(x) + R(x), cu deg R(x) < deg D(x).

  • 30. Schema lui Horner

    Metodă: se folosește pentru evaluarea eficientă a polinoamelor și pentru împărțire cu rest.

  • 31. Divizibilitatea polinoamelor. Teorema lui Bezout

    Formulă: P(x) are x – a ca factor dacă și numai dacă P(a)=0.

  • 33. Polinoame reductibile și ireductibile. Descompunerea polinoamelor

    Concept: un polinom este ireductibil dacă nu se poate scrie ca produsul a doi polinoame de grad mai mic.

  • 36. Relațiile lui Viète

    Formule: pentru ax²+bx+c=0, x₁+x₂ = −b/a și x₁·x₂ = c/a.

  • 38. Rădăcinile polinoamelor cu coeficienți reali. Partea I

    Concept: se determină semnul și numărul rădăcinilor în funcție de discriminant.

  • 39. Rădăcinile polinoamelor cu coeficienți reali. Partea II

    Continuare: aplicarea relațiilor lui Viète.

  • 40. Ecuații reciproce de gradul 3 și 4

    Concept: ecuații simetrice în coeficienți, ce pot fi reduse prin substituție.

Capitolul Doi – Analiză

  • 41. Derivata unei funcții. Formule de derivare

    Formule: f'(x) = limₕ→0 [f(x+h) – f(x)]/h; reguli: derivata sume, produs, lanț, etc.

  • 42. Formule de derivare. Funcții trigonometrice

    Exemple: (sin x)' = cos x, (cos x)' = -sin x, (tan x)' = sec² x.

  • 43. Derivarea funcțiilor compuse

    Regula lanțului: (f∘g)'(x) = f'(g(x))·g'(x).

  • 45. Primitive ale unei funcții

    Concept: F este primitivă a lui f dacă F' = f.

  • 46. Existența primitivelor

    Teorema fundamentală a analizei: dacă f este continuă pe [a, b], atunci are primitivă.

  • 48. Operații cu funcții care admit primitive

    Concept: suma, produsul și scalarul de primitive sunt primitive ale operațiilor corespunzătoare.

  • 49. Primitive uzuale – Partea I

    Exemple: ∫ x^n dx = x^(n+1)/(n+1), n ≠ -1.

  • 50. Primitive uzuale – Partea II

    Continuare: primitive ale funcțiilor trigonometrice, exponentiale etc.

  • 51. Primitive uzuale – Partea III

    Concept: liste de primitive standard.

  • 53. Metoda integrării prin părți – Partea I

    Formulă: ∫ u dv = u·v − ∫ v du.

  • 54. Metoda integrării prin părți – Partea II

    Continuare: aplicarea metodei integrării prin părți.

  • 55. Integrala definită. Formula Leibniz-Newton

    Formulă: ∫ₐᵇ f(x) dx = F(b) − F(a), unde F este primitiva lui f.

  • 58. Proprietăți ale integralei definite – Partea I

    Concepte: aditivitate, linearitate.

  • 59. Proprietăți ale integralei definite – Partea II

    Continuare: relații între integrale.

  • 61. Integrare prin părți în calculul integralelor definite

    Aplicarea metodei integrării prin părți la integrale definite.

  • 62. Integrarea prin schimbare de variabilă – Partea I

    Formulă: ∫ f(g(x))·g'(x) dx = ∫ f(u) du, cu u = g(x).

  • 63. Integrarea prin schimbare de variabilă – Partea II

    Continuare: aplicarea schimbării de variabilă.

  • 64. Integrarea prin schimbare de variabilă – Partea III

    Concept: tehnici suplimentare de schimbare de variabilă.

  • 65. Integrarea funcțiilor raționale – Partea I
