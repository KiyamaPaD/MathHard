Clasa X

Această pagină prezintă formulele și conceptele esențiale din materialul didactic pentru Clasa X.

Capitolul Unu – Mulțimi de numere

  • 1. Radicalul de ordin n

    Formulă: √[n](a) reprezintă rădăcina n-a a lui a.

  • 2. Radicalul impar din numere negative

    Concept: pentru n impar, √[n](a) este definit și pentru a < 0, cu relația √[n](a) = -√[n](-a).

  • 3. Proprietăți ale radicalilor

    Exemple: √[n](ab) = √[n](a)·√[n](b), √[n](aⁿ) = |a| (dacă n este par) sau a (dacă n este impar).

  • 4. Aproximări raționale ale radicalilor de ordin n

    Concept: se pot aproxima radicalii prin fracții, de exemplu prin metoda lui Newton.

  • 5. Radicali. Raționalizarea numitorilor

    Exemplu: a/√b = (a√b)/b.

  • 7. Puteri cu exponent negativ și rațional

    Formulă: a^(-p/q) = 1/(a^(p/q)).

  • 8. Puteri cu exponent real

    Concept: a^(r), unde r ∈ ℝ.

  • 10. Logaritmi din numere reale pozitive

    Definiție: logₐ(b) este numărul x astfel încât aˣ = b, pentru b > 0, a > 0 și a ≠ 1.

  • 11. Proprietățile logaritmilor

    Exemple: logₐ(xy) = logₐ(x) + logₐ(y); logₐ(x/y) = logₐ(x) − logₐ(y); logₐ(x^r) = r·logₐ(x).

  • 12. Schimbarea bazei unui logaritm

    Formulă: log_b(a) = log_c(a) / log_c(b), pentru orice bază c > 0, c ≠ 1.

  • 13. Logaritmarea unei expresii

    Concept: aplicarea logaritmilor pentru simplificarea expresiilor algebrice.

  • 15. Numerele complexe. Forma algebrică

    Formulă: z = a + bi, unde a, b ∈ ℝ și i² = -1.

  • 16. Operații algebrice cu numere complexe

    Exemple: (a+bi)+(c+di) = (a+c)+(b+d)i; (a+bi)·(c+di) = (ac−bd)+(ad+bc)i.

  • 17. Conjugatul unui număr complex

    Formulă: conjugatul lui z = a + bi este a - bi.

  • 18. Modulul unui număr complex

    Formulă: |z| = √(a² + b²).

  • 19. Interpretarea geometrică a numerelor complexe

    Concept: reprezentarea lui z = a + bi în planul complex (axa reală și imaginară).

  • 20. Operații cu numere complexe. Interpretarea geometrică

    Concept: operațiile de adunare și înmulțire se interpretează ca translații și rotații în plan.

  • 22. Rezolvarea în C a ecuațiilor de gradul doi cu coeficienți reali

    Formulă: x = [–b ± √(b² – 4ac)]/(2a) pentru ecuația ax²+bx+c=0.

  • 23. Ecuații bipatrate

    Concept: se substituie y = x² pentru a transforma ecuația în una de gradul II, apoi se rezolvă.

Capitolul Doi – Funcții și ecuații

  • 24. Recapitulare funcții

    Concept: o funcție f: A → B asociază fiecărui x ∈ A un singur y ∈ B.

  • 25. Analizarea unui grafic de funcție. Partea I

    Concept: identificarea punctelor cheie ale graficului (intersecții, maxim, minim).

  • 26. Analizarea unui grafic de funcție. Partea II

    Concept: interpretarea variațiilor funcției pe baza graficului.

  • 28. Funcții injective

    Definiție: f este injectivă dacă f(x₁) = f(x₂) implică x₁ = x₂.

  • 29. Funcții surjective

    Definiție: f este surjectivă dacă pentru orice y ∈ B există x ∈ A astfel încât f(x) = y.

  • 30. Funcții bijective

    Definiție: f este bijectivă dacă este atât injectivă, cât și surjectivă.

  • 33. Inversa unei funcții

    Definiție: f⁻¹ este funcția inversă a lui f dacă f(f⁻¹(y)) = y, ∀y ∈ B.

  • 35. Funcția putere cu exponent natural

    Formulă: f(x) = xⁿ, unde n ∈ ℕ.

  • 36. Funcția radical de ordin n

    Formulă: f(x) = √[n](x).

  • 37. Ecuații iraționale ce conțin radicali

    Concept: ecuații care includ termeni de forma √[n](expression); se ridică la putere pentru eliminare.

  • 38. Ecuații iraționale cu mai mulți radicali

    Concept similar: se izolează radicalii și se ridică la putere.

  • 39. Funcția exponențială

    Formulă: f(x) = a^x, de obicei f(x) = e^x pentru a = e.

  • 40. Ecuații exponențiale. Partea I

    Concept: ecuații de forma a^x = b, se aplică logaritmi.

  • 41. Ecuații exponențiale. Partea II

    Continuare: rezolvarea ecuațiilor exponențiale.

  • 42. Funcția logaritmică

    Formulă: f(x) = logₐ(x), unde a > 0, a ≠ 1.

  • 43. Ecuații logaritmice. Partea I

    Formulă: logₐ(x) = b ⟹ x = a^b.

  • 44. Ecuații logaritmice. Partea II

    Continuare: rezolvarea ecuațiilor logaritmice.

  • 45. Creșterea exponențială

    Concept: funcția exponențială este strict crescătoare pentru bază > 1.

  • 46. Descreșterea exponențială

    Concept: funcția exponențială este descrescătoare pentru 0 < bază < 1.

  • 47. Creștere și descreștere logaritmică

    Concept: funcția logaritmică este crescătoare dacă baza > 1, descrescătoare dacă 0 < baza < 1.

  • 48. Funcții trigonometrice. Introducere

    Definiții de bază pentru sin, cos, tan etc.

  • 49. Funcțiile sinus și arcsinus

    Formule: sin(x); arcsin(x) este funcția inversă a sinusului, definită pe [-1,1].

  • 50. Funcțiile cosinus și arccosinus

    Formule: cos(x); arccos(x) este funcția inversă a cosinusului, definită pe [-1,1].

  • 51. Funcțiile tangentă și arctangentă

    Formule: tan(x)= sin(x)/cos(x); arctan(x) este inversa tangentului.

  • 52. Funcțiile cotangentă și arccotangentă

    Formule: cot(x)= cos(x)/sin(x); arccot(x) este inversa cotangentului.

  • 53. Valori trigonometrice importante

    Exemple: sin(30°)=0.5, cos(60°)=0.5, tan(45°)=1, etc.

  • 54. Valori trigonometrice inverse

    Concept: determinarea unghiurilor din valorile sin, cos, etc.

  • 57. Ecuații trigonometrice. Partea I

    Exemplu: sin(x)= a, se rezolvă pentru x.

  • 58. Ecuații trigonometrice. Partea II

    Continuare: rezolvarea ecuațiilor trigonometrice complexe.

Capitolul Trei – Metode de numărare

  • 59. Metoda inducției matematice

    Principiul inducției: dacă P(1) este adevărat și P(k) → P(k+1), atunci P(n) este adevărat ∀ n.

  • 60. Numărul de funcții

    Concept: numărul de funcții de la o mulțime A la o mulțime B este |B|^|A|.

  • 61. Permutări

    Formulă: numărul de permutări ale unei mulțimi de n elemente este n!.

  • 64. Aranjamente și combinări

    Formule: A(n, k)= n!/(n-k)! și C(n, k)= n!/(k!(n-k)!).

  • 67. Proprietățile combinărilor

    Concept: simetria combinărilor: C(n, k) = C(n, n-k).

  • 68. Probleme de numărare

    Concept: aplicarea principiilor de numărare pentru a determina numărul de aranjamente.

  • 69. Binomul lui Newton

    Formulă: (a+b)^n = Σ (n choose k) a^(n-k) b^k, pentru k=0,...,n.

Capitolul Patru – Matematici financiare

  • 72. Procente

    Formulă: procent = (parte / întreg) × 100%.

  • 73. Scumpiri și ieftiniri

    Concept: calculul modificărilor procentuale.

  • 74. Dobânda simplă și compusă

    Formule: dobânda simplă: I = P · r · t; dobânda compusă: A = P(1 + r)^t.

  • 75. Constituirea de capital. Amortizarea unei datorii

    Concept: calculul ratelor de amortizare.

  • 76. Taxa pe Valoare Adăugată (T.V.A.)

    Formulă: TVA = preț × (TVA% / 100).

  • 77. Cost de producție și profit

    Concept: diferența dintre cost și prețul de vânzare.

  • 78. Culegerea și clasificarea datelor statistice

    Concept: organizarea datelor în funcție de categorii.

  • 79. Prelucrarea și reprezentarea datelor statistice

    Concept: reprezentarea datelor prin grafice și tabele.

  • 80. Parametri de poziție: medii, dispersie, abateri

    Formule: medie = (∑ xi)/n; deviație standard, etc.

  • 81. Probabilități – Evenimente egal probabile

    Formulă: P(E) = număr cazuri favorabile / număr total cazuri.

  • 83. Variabile aleatoare

    Concept: variabila aleatoare atribuie un număr fiecărui eveniment.

Capitolul Cinci – Geometrie

  • 84. Distanța dintre două puncte în plan

    Formulă: d = √[(x₂ − x₁)² + (y₂ − y₁)²].

  • 85. Ecuația explicită a unei drepte

    Formulă: y = mx + c.

  • 86. Ecuația unei drepte determinate de un punct și o direcție

    Formulă: y − y₀ = m(x − x₀).

  • 87. Ecuația unei drepte determinate de două puncte

    Formulă: (y − y₁)/(x − x₁) = (y₂ − y₁)/(x₂ − x₁).

  • 88. Condiția de paralelism

    Condiție: m₁ = m₂ (egalitatea pantelor).

  • 89. Condiția de perpendicularitate

    Condiție: m₁ × m₂ = −1.

  • 91. Intersecția a două drepte în plan

    Concept: se rezolvă sistemul de ecuații al drepților.

  • 92. Distanțe în plan

    Exemplu: distanța de la un punct (x₀,y₀) la dreaptă Ax + By + C = 0: d = |Ax₀+By₀+C|/√(A²+B²).

  • 93. Arii în plan

    Formule pentru triunghi: A = ½·base·height, pentru dreptunghi: A = l×L etc.

  • 94. Vectori de poziție. Operații

    Concept: adunare, scădere, produs scalar (u·v = |u||v|cosθ).

  • 95. Vectori în plan

    Definiție: un vector v = (x, y).
