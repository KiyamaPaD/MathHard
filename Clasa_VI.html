Clasa a VI-a

Aici vor fi formulele din clasa a VI-a din matematică.

Algebră

1. Mulțimea numerelor naturale:

  • ℕ = {0, 1, 2, 3, …}
  • (uneori se consideră ℕ* = {1, 2, 3, …})

2. Mulțimi:

  • Notare: x ∈ A (elementul x aparține mulțimii A)
  • Definiție: A = {x | x satisface proprietatea P(x)}
  • Relație de submulțime: A ⊆ B

3. Relații între mulțimi:

  • Egalitate: A = B
  • Submulțime: A ⊆ B
  • Mulțimi disjuncte: A ∩ B = ∅

4. Operații cu mulțimi:

  • Intersecție:
    A ∩ B = {x | x ∈ A și x ∈ B}
  • Reuniune:
    A ∪ B = {x | x ∈ A sau x ∈ B}
  • Diferență:
    A \ B = {x | x ∈ A și x ∉ B}
  • Diferență simetrică:
    A Δ B = (A \ B) ∪ (B \ A)

5. Numere prime. Numere compuse:

  • Număr prim: p > 1 și, pentru orice divizor d ∈ ℕ, dacă d | p atunci d = 1 sau d = p
  • Număr compus: n > 1 și ∃ a, b ∈ ℕ, 1 < a, b < n astfel încât n = a · b

6. Descompunerea numerelor naturale în produs de factori primi:

Factorizare:
n = p₁α₁ · p₂α₂ · … · pkαk (unde p₁, p₂, …, pk sunt numere prime)

7. Cel mai mare divizor comun (c.m.m.d.c.). Numere prime între ele:

  • Notare: cmmdc(a, b)
  • Dacă a și b sunt prime între ele: cmmdc(a, b) = 1

8. Cel mai mic multiplu comun (c.m.m.m.c.):

Relație:
cmmmc[a, b] = (a · b) / cmmdc(a, b)

9. Proprietăți ale divizibilității în mulțimea numerelor naturale:

  • Definiție: a | b dacă și numai dacă ∃ k ∈ ℕ astfel încât b = a · k
  • Proprietate: Dacă a | b și b | c, atunci a | c

10. Rapoarte:

  • Raportul dintre a și b: a : b sau a/b

11. Proporții:

  • Proporție: a : b = c : d sau a/b = c/d
  • Formula de bază: a/b = c/d ⟹ a · d = b · c
  • Proporții derivate: dacă a : b = c : d, se pot deriva relații de forma (a + c) : (b + d) = a : b (în anumite contexte)

12. Șir de rapoarte egale:

Dacă a₁ : a₂ = a₂ : a₃ = … = aₙ₋₁ : aₙ = k, atunci: ai+1 = k · ai, pentru i = 1, 2, …, n – 1

13. Mărimi direct proporționale:

Relație: y ∝ x ⟹ y = k · x (unde k este constanta de proporționalitate)

14. Mărimi invers proporționale:

Relație: y ∝ 1/x ⟹ y = k/x

15. Regula de trei simplă:

Dacă a : b = c : x, atunci: x = (b · c) / a

16. Elemente de organizare a datelor:

  • Medie: μ = (∑ xi) / n
  • Mediana: (valoarea mijlocie în setul ordonat)
  • Mod: (cea mai frecventă valoare)

17. Probabilități:

  • Probabilitate: P(E) = (număr cazuri favorabile) / (număr total cazuri)
  • Complement: P(E') = 1 − P(E)

18. Mulțimea numerelor întregi:

ℤ = { …, −3, −2, −1, 0, 1, 2, 3, … }

19. Adunarea numerelor întregi

Exemple:
(−a) + (−b) = −(a + b)
(−a) + b = b − a

20. Adunarea numerelor întregi. Proprietăți:

  • Comutativitate: a + b = b + a
  • Asociativitate: (a + b) + c = a + (b + c)
  • Element neutru: a + 0 = a

21. Scăderea numerelor întregi:

Relație: a − b = a + (−b)

22. Înmulțirea numerelor întregi:

  • Reguli de semn:
    (−a) · (−b) = a · b
    (−a) · b = −(a · b)

22. Înmulțirea numerelor întregi. Proprietăți:

  • Comutativitate: a · b = b · a
  • Asociativitate: (a · b) · c = a · (b · c)
  • Element neutru: a · 1 = a

23. Împărțirea numerelor întregi:

Relație: a : b = a/b, cu b ≠ 0

24. Puterea cu exponent număr natural a unui număr întreg nenul:

Definiție:
aⁿ = a · a · … · a (n factori)

Proprietăți:
aᵐ · aⁿ = a^(m+n)
(aᵐ)ⁿ = a^(m·n)
(a · b)ⁿ = aⁿ · bⁿ

25. Ecuații:

Ecuație liniară:
a·x + b = 0 ⟹ x = −b/a, cu a ≠ 0

26. Inecuații în ℤ:

Exemple:
a·x + b > 0,
a·x + b < 0
(Notă: la multiplicare sau împărțire cu numere negative se inversează semnul inegalității)

27. Număr rațional. Mulțimea numerelor raționale:

Definiție:
ℚ = {a/b | a ∈ ℤ, b ∈ ℤ \ {0}}

28. Adunarea și scăderea numerelor raționale:

  • Adunare: a/b + c/d = (a·d + b·c) / (b·d)
  • Scădere: a/b − c/d = (a·d − b·c) / (b·d)

29. Înmulțirea și împărțirea numerelor raționale:

  • Înmulțire: (a/b) · (c/d) = (a·c) / (b·d)
  • Împărțire: (a/b) : (c/d) = (a·d) / (b·c), cu c ≠ 0

30. Puterea cu exponent număr întreg a unui număr rațional nenul:

Formula:
(a/b)ⁿ = aⁿ / bⁿ, cu b ≠ 0

31. Ordinea efectuării operațiilor și folosirea parantezelor:

Ordinea operațiilor (PEMDAS/BODMAS):
Paranteze → Exponenți → Înmulțire și Împărțire → Adunare și Scădere

32. Ecuații în mulțimea numerelor raționale:

Ecuații de forma: (a/b)·x + (c/d) = 0
Rezolvare similară cu ecuațiile din ℤ

33. Unghiuri suplementare. Unghiuri complementare:

  • Unghiuri complementare: A + B = 90°
  • Unghiuri suplementare: A + B = 180°

34. Unghiuri adiacente. Bisectoarea unui unghi:

Două unghiuri adiacente: A + B = unghiul total
Bisectoare: Dacă ∠ABC este bisectat, atunci: ∠ABD = ∠DBC = ½ · ∠ABC

35. Unghiuri opuse la vârf. Unghiuri formate în jurul unui punct:

Unghiuri opuse la vârf sunt egale
Suma unghiurilor în jurul unui punct: 360°

36. Drepte perpendiculare în plan. Mediatoarea unui segment.

Drepte perpendiculare: d₁ ⊥ d₂
Mediatoarea unui segment: Dreapta care trece prin mijlocul segmentului și este perpendiculară pe acesta

37. Drepte paralele. Axioma paralelelor.

Drepte paralele: d₁ ∥ d₂
Axioma paralelelor: Prin orice punct exterior unei drepte se poate trasa o singură dreaptă paralelă cu aceasta

38. Criterii de paralelism.

Dacă o transversala intersectează două drepte și unghiurile corespondente sunt egale, atunci dreptele sunt paralele

39. Cerc. Elemente în cerc.

  • Diametru: d = 2r
  • Circumferință: C = 2πr
  • Arie: A = πr²

40. Pozițiile unei drepte față de un cerc. Pozițiile relative ale două cercuri.

Relații:
Dacă distanța de la centru la dreaptă (d) > r, dreapta este externă
Dacă d = r, dreapta este tangentă
Dacă d < r, dreapta intersectează cercul

41. Triunghiul. Clasificare. Perimetru.

Perimetru triunghi: P = a + b + c

42. Suma măsurilor unghiurilor unui triunghi.

Suma unghiurilor: A + B + C = 180°

43. Construcția triunghiurilor.

(Tehnici de construcție – nu există formule fixe)

44. Linii importante în triunghi.

Mediana: segmentul care unește un vârf cu mijlocul laturii opuse
Bisectoarea: segmentează un unghi în două unghiuri egale
Înălțimea: segment perpendicular dintr-un vârf pe latura opusă

45. Congruența triunghiurilor oarecare.

Criterii de congruență:
Latură-Unghi-Latură (LUL)
Unghi-Latură-Unghi (ULU) Latură-Latură-Latură (LLL)

46. Proprietăți ale triunghiului isoscel.

Într-un triunghi isoscel: Două laturi sunt egale (a = b) și unghiurile la bază sunt egale.

47. Proprietăți ale triunghiului echilateral.

Triunghi echilateral: a = b = c, iar toate unghiurile sunt de 60°.

48. Proprietăți ale triunghiului dreptunghic.

Teorema lui Pitagora: a² + b² = c²
(unde a și b sunt catetele, iar c este ipotenuza)

Aria triunghiului dreptunghic: A = (a · b) / 2
