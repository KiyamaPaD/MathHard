Clasa a V-a

Aici vor fi formulele din clasa a V-a din matematică.

Algebră

1. Proprietăți Fundamentale:

  • Proprietatea comutativă a adunării:
    a + b = b + a
  • Proprietatea comutativă a înmulțirii:
    a × b = b × a
  • Proprietatea asociativă a adunării:
    (a + b) + c = a + (b + c)
  • Proprietatea asociativă a înmulțirii:
    (a × b) × c = a × (b × c)
  • Proprietatea distributivă:
    a × (b + c) = a × b + a × c

2. Formule de Desfacere și Împachetare:

  • Factor comun:
    a × b + a × c = a × (b + c)
  • Deschiderea parantezelor (distributivitate):
    a × (b + c) = a × b + a × c

3. Suma lui Gauss:

1 + 2 + 3 + ... + n = n × (n + 1) / 2

4. Operații cu Exponenți:

  • Înmulțirea puterilor cu aceeași bază:
    am × an = am + n
  • Împărțirea puterilor cu aceeași bază (a ≠ 0):
    am / an = am − n
  • Puterea unei puteri:
    (am)n = am × n
  • 0 la exponent:
    a0 = 1
  • 1 la exponent:
    a1 = a
  • 0 la baza:
    0a = 0
  • 1 la baza:
    1a = 1

5. Divizibilitatea numerelor naturale:

  • Formula de legătură între CMMDC și CMMMC:
    CMMMC(a, b) = (a × b) / CMMDC(a, b)
  • Divizibilitate cu 2:
    Un număr este divizibil cu 2 dacă ultima cifră este pară (0, 2, 4, 6, 8).
  • Divizibilitate cu 3:
    Un număr este divizibil cu 3 dacă suma cifrelor sale este divizibilă cu 3.
  • Divizibilitate cu 4:
    Un număr este divizibil cu 4 dacă numărul format de ultimele două cifre este divizibil cu 4.
  • Divizibilitate cu 5:
    Un număr este divizibil cu 5 dacă ultima cifră este 0 sau 5.
  • Divizibilitate cu 6:
    Un număr este divizibil cu 6 dacă este divizibil atât cu 2, cât și cu 3.
  • Divizibilitate cu 8:
    Un număr este divizibil cu 8 dacă numărul format de ultimele trei cifre este divizibil cu 8.
  • Divizibilitate cu 9:
    Un număr este divizibil cu 9 dacă suma cifrelor sale este divizibilă cu 9.
  • Divizibilitate cu 10:
    Un număr este divizibil cu 10 dacă ultima cifră este 0.
  • Divizibilitate cu 11:
    Un număr este divizibil cu 11 dacă diferența dintre suma cifrelor de pe poziții impare și suma cifrelor de pe poziții pare este 0 sau un multiplu de 11.

6. Fracții Ordinare:

  • Echivalența fracțiilor:
    Două fracții a/b și c/d sunt echivalente dacă: a × d = b × c
  • Amplificarea și simplificarea fracțiilor:
    Amplificare: a/b = (a×k)/(b×k), pentru orice k ≠ 0.
    Simplificare: a/b se reduce la (a÷k)/(b÷k) dacă k este un divizor comun al lui a și b.
  • Adunarea și scăderea fracțiilor:
    Dacă fracțiile au același numitor: a/b + c/b = (a + c)/b și a/b - c/b = (a - c)/b.
    Dacă fracțiile au numitori diferiți: a/b + c/d = (a×d + c×b)/(b×d) și a/b - c/d = (a×d - c×b)/(b×d).
  • Înmulțirea și împărțirea fracțiilor:
    Înmulțirea: (a/b) × (c/d) = (a×c)/(b×d).
    Împărțirea: (a/b) ÷ (c/d) = (a×d)/(b×c), cu c/d ≠ 0.
  • Aflarea unei fracții dintr-un număr:
    Pentru a calcula a/b din n: (a/b) × n.

7. Fracții Zecimale:

  • Orice fracție zecimală poate fi transformată într-o fracție ordinara (ex.: 0,25 = 25/100 = 1/4).
  • Un număr natural se poate scrie ca fracție zecimală (ex.: 5 = 5,0).
  • Operații: Adunarea și scăderea se realizează prin alinierea virgulilor, înmulțirea se efectuează tratând numerele ca întregi (apoi se plasează virgula în rezultat), iar împărțirea se tratează ca operație cu numere zecimale, deplasând virgula dacă este necesar.

8. Fracții zecimale periodice:

  • Notare: Se marchează partea repetitivă cu o bară deasupra (ex.: 0,3̅ pentru 0,333…).
  • Transformarea în fracție ordinara:
    Dacă x = 0,a(b̅) (unde a este partea nemodificată și b partea repetitivă), atunci:
    x = (numărul format din a și b – numărul format doar din a) / (10^(n+m) – 10^n),
    unde n = numărul de cifre din a și m = numărul de cifre repetate.
    Exemplu simplificat: 0,3̅ = 3/9 = 1/3.

Geometrie

1. Mijlocul unui segment:

Dacă A și B sunt capetele segmentului, punctul M este mijlocul dacă: AM = MB.

2. Simetricul unui punct:

Pentru un punct P și un punct de referință O, simetricul lui P (notat P') este definit astfel încât O să fie mijlocul segmentului PP'.

3. Unități de măsură pentru lungime:

  • 1 kilometru (km) = 10 hectometrii (hm) = 102 decametrii (dam) = 103 metrii (m)
  • 1 metru (m) = 10 decimetrii (dm) = 102 centimetrii (cm) = 103 milimetrii (mm)

4. Unități de măsură pentru arie:

  • 1 kilometru pătrat (km2) = 102 hectometrii pătrați (hm2) = 104 decametrii pătrați (dam2) = 106 metrii pătrați (m2)
  • 1 metru pătrat (m2) = 102 decimetrii pătrați (dm2) = 104 centimetrii pătrați (cm2) = 106 milimetrii pătrați (mm2)
  • 1 arie (ar) = 102 metrii pătrați
  • 1 hectar (ha) = 102 arii (ar) = 104 metrii pătrați (m2)

5. Unități de măsură pentru volum:

  • 1 kilometru cub (km3) = 103 hectometrii cubi (hm3) = 106 decametrii cubi (dam3) = 109 metrii cubi (m3)
  • 1 metru cub (m3) = 103 decimetrii cubi (dm3) = 106 centimetrii cubi (cm3) = 109 milimetrii cubi (mm3)
  • 1 metru cub (m3) = 103 litri (L)
  • 1 litru (L) = 1 decimetru cub (dm3) = 103 centimetrii cubi (cm3)

6. Unități de măsură pentru unghi:

  • 1 grad (°) = 60 minute (') = 3600 secunde (")
  • 1 minut (') = 60 secunde (")

7. Unghiuri și operații:

  • Tipuri de unghiuri: Unghi nul: 0°; Unghi drept: 90°; Unghi acut: < 90°; Unghi obtuz: > 90° și < 180°; Unghi reflex: > 180° și < 360°; Unghi complet: 360°
  • Congruența unghiurilor: <ABC ≡ <MNP, dacă măsura unghiului ABC este egală cu măsura unghiului MNP.

8. Perimetru și arie:

  • Pătrat: Arie: A = l², Perimetru: P = 4 × l
  • Dreptunghi: Arie: A = l × L, Perimetru: P = 2 × (l + L)
  • Triunghi: Perimetru: P = a + b + c

9. Volum:

  • Cub: Volum: V = l³
  • Paralelipiped dreptunghic: Volum: V = l × L × h
