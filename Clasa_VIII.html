Clasa a VIII-a

Aici vor fi formulele de bază și conceptele esențiale din materialul didactic pentru Clasa a VIII-a.

Capitolul Unu – Mulțimea numerelor reale

  • 1. Numere raționale, numere iraționale

    Definiție: Numerele raționale se pot scrie ca fracții; numerele iraționale nu pot fi exprimate exact prin fracții.

  • 2. Aproximarea, rotunjirea și compararea numerelor reale

    Concepte: Rotunjirea se face conform regulilor standard; comparațiile se stabilesc folosind ≤, ≥.

  • 3. Axa numerelor reale. Opusul și modulul unui număr real

    Formula: Opusul lui a este −a; Modulul: |a| = a dacă a ≥ 0, |a| = −a dacă a < 0.

  • 4. Mulțimi definite printr-o proprietate comună

    Formulă: A = {x | P(x) este adevărat}

  • 5. Operații cu mulțimi

    Exemple: A ∪ B, A ∩ B, A \ B, A Δ B

  • 6. Intervale

    Exemple: [a, b], (a, b), [a, b), (a, b]

  • 7. Operații cu intervale

    Concept: Uniunea și intersecția intervalelor.

  • 8. Inecuații de forma ax+b ≥ 0 (și variante)

    Exemplu: ax + b ≥ 0 se rezolvă astfel: x ≥ −b/a (dacă a > 0; invers dacă a < 0).

  • 10. Operații cu numere reale

    Operații de bază: adunare, scădere, înmulțire și împărțire (cu b ≠ 0).

  • 11. Operații cu numere reale – Proprietăți

    Proprietăți: comutativitate, asociativitate și distributivitate.

  • 12. Ridicarea la putere cu exponent număr întreg

    Formulă: aⁿ, unde n ∈ ℤ

  • 13. Proprietăți ale relației de inegalitate

    Exemplu: dacă a < b și b < c, atunci a < c.

  • 14. Alte proprietăți ale modulului. Eroarea absolută

    Concept: eroarea absolută se exprimă prin |a − b|.

  • 15. Partea întreagă și partea fracționară

    Formule: partea întreagă: ⌊a⌋; partea fracționară: {a} = a − ⌊a⌋.

  • 16. Calcule cu radicali. Raționalizarea numitorului

    Exemplu: a/√b = (a√b)/b.

Capitolul Doi – Calcul algebric

  • 17. Calcule cu numere reale reprezentate prin litere

    Concept: se efectuează operații algebrice standard (adunare, înmulțire etc.).

  • 18. Formule de calcul prescurtat

    Exemplu: (a+b)² = a² + 2ab + b²; (a−b)² = a² − 2ab + b².

  • 21. Metode de descompunere în factori

    Exemplu: a² − b² = (a − b)(a + b).

  • 22. Descompunerea în factori prin metoda grupării

    Concept: se grupează termeni pentru a factoriza comun.

  • 24. Descompuneri în factori cu formule de calcul prescurtat

    Exemplu: (a+b)³ = a³ + 3a²b + 3ab² + b³.

  • 25. Operații cu rapoarte de numere reale

    Concept: se operează cu expresii de forma a/b.

Capitolul Trei – Ecuații și inecuții

  • 28. Ecuații de forma a² + bx + c = 0 (cazul general)

    Formula: x = [–b ± √(b² – 4ac)]/(2a)

  • 29. Ecuații reductibile la a² + bx + c = 0

    Concept: se aduce ecuația la forma standard și se aplică formula generală.

Capitolul Patru – Funcții

  • 32. Produsul cartezian a două mulțimi

    Formulă: A × B = {(a, b) | a ∈ A, b ∈ B}

  • 33. Funcții

    Definiție: o funcție f: A → B asociază fiecărui element din A un element din B.

  • 35. Graficul unei funcții

    Concept: graficul funcției este mulțimea punctelor {(x, f(x)) | x ∈ A}.

  • 37. Reprezentarea grafică a funcției f(x) = ax + b

    Formulă: ecuația liniei drepte y = ax + b.

  • 38. Reprezentarea grafică a funcției f: I → ℝ, f(x) = ax + b

    Concept: linie dreaptă pentru x ∈ I, unde I este un interval din ℝ.

Capitolul Cinci – Elemente de statistică

  • 43. Sortarea și organizarea datelor – Partea I

    Concept: clasificarea datelor după tip și frecvență.

  • 44. Sortarea și organizarea datelor – Partea II

    Continuare: organizarea datelor conform frecvenței absolute.

  • 46. Reprezentarea geometrică a seriilor statistice

    Exemplu: histograma sau diagrama cu bare.

  • 47. Indicatorii tendinței centrale

    Formule: medie, mediană, mod.

Capitolul Șase – Elemente de geometrie în spațiu

  • 50. Determinarea planului

    Formulă: Ax + By + Cz + D = 0.

  • 51. Poziții relative ale unei drepte față de un plan

    Concept: dreapta poate fi paralelă, secantă sau perpendiculară pe un plan.

  • 52. Pozițiile relative a două plane

    Concept: plane paralele sau intersectante.

  • 53. Paralelism în spațiu

    Condiție: direcțiile identice ale dreptei.

  • 54. Alte teoreme de paralelism

    Condiții suplimentare pentru paralelism în spațiu.

  • 55. Secțiuni paralele cu baza în prismă/cilindru

    Concept: secțiuni similare cu baza, proporționale cu aceasta.

  • 56. Secțiuni paralele cu baza în piramidă

    Formula: arii proporționale cu pătratul raportului distanței față de vârf.

  • 57. Secțiuni paralele cu baza în con

    Concept: arii proporționale cu pătratul raportului înălțimii.

  • 58. Măsura unghiului dintre două drepte

    Formulă: θ = cos⁻¹[(u·v)/(|u||v|)].

  • 59. Dreaptă perpendiculară pe un plan

    Concept: perpendicularitate față de toate dreptele din plan la punctul de intersecție.

  • 60. Distanța de la un punct la un plan

    Formulă: d = |Ax₀ + By₀ + Cz₀ + D| / √(A²+B²+C²).

  • 61. Distanța dintre două plane paralele

    Formulă: d = |D₂ − D₁| / √(A²+B²+C²).

  • 62. Înălțimea

    Concept: măsurată perpendicular pe baza corpului.

  • 63. Plane perpendiculare

    Concept: plane ale căror normale sunt perpendiculare.

  • 64. Secțiuni diagonale/axiale

    Descrierea secțiunilor în corpuri (diagonale sau axiale).

  • 65. Secțiuni axiale

    Reprezentarea secțiunilor axiale.

  • 66. Proiecții ortogonale ale unui punct pe o dreaptă și pe un plan – Partea I

    Formula standard de proiecție ortogonală.

  • 67. Proiecții ortogonale – Partea II

    Extinderea formulei de proiecție la drepte și segmente.

  • 68. Unghiul dintre o dreaptă și un plan

    Formulă: θ = 90° − φ, unde φ este unghiul dintre dreaptă și normală la plan.

  • 69. Unghi diedru

    Concept: unghiul dintre două plane intersectante.

  • 70. Unghiul dintre două plane

    Formulă: unghiul dintre plane = unghiul dintre normalele lor.

  • 71. Teorema celor trei perpendiculare

  • 72. Reciproca I a teoremei celor trei perpendiculare

    Condiții reciproce.

  • 73. Reciproca II a teoremei celor trei perpendiculare

Capitolul Șapte – Arii și volume ale unor corpuri geometrice

  • 74. Calcularea unor distanțe pe fețele sau în interiorul corpurilor

    Concept: expresii geometrice pentru distanțe.

  • 75. Alte distanțe în corpurile studiate

    Metode suplimentare de măsurare a distanțelor.

  • 76. Calcularea unor măsuri de unghiuri în corpuri

    Formule pentru determinarea unghiurilor în interiorul corpurilor.

  • 77. Aria prismei drepte

    Formulă: Aria = Aria bazei × înălțime.

  • 78. Volumul unei prisme drepte

    Formulă: Volum = Aria bazei × înălțime.

  • 79. Aria laterală și totală a piramidei regulate

    Formulă: Arie totală = Aria bazei + Aria laterală.

  • 80. Volumul piramidei regulate

    Formulă: Volum = (Aria bazei × înălțime) / 3.

  • 81. Aria trunchiului de piramidă regulată

    Formulă: Aria trunchi = ((A₁ + A₂)/2) × înălțime.

  • 82. Volumul trunchiului de piramidă

    Formulă: Volum = (A₁ + A₂ + √(A₁·A₂)) × înălțime / 3.

  • 83. Aria laterală și totală a cilindrului drept

    Formule: Aria laterală = 2πr·h; Arie totală = 2πr(h + r).

  • 84. Volumul cilindrului drept

    Formulă: Volum = πr²h.

  • 85. Aria laterală și totală a conului

    Formule: Aria laterală = πr·l; Arie totală = πr(r + l), unde l este apotema.

  • 86. Volumul conului

    Formulă: Volum = (πr²h)/3.

  • 87. Aria laterală și totală a trunchiului de con circular drept

    Formule: Aria laterală = π(R + r)l; Arie totală = π[(R + r)l + R² + r²].

  • 88. Volumul trunchiului de con circular drept

    Formulă: Volum = (πh/3)(R² + Rr + r²).

  • 89. Aria sferei

    Formulă: Arie = 4πr².

  • 90. Volumul sferei

    Formulă: Volum = (4/3)πr³.
